## 📝 Описание

**Wordy Fresh** — умная и современная версия классической word-головоломки. Угадывай слова разной длины, узнавай их значения прямо в игре, соревнуйся с друзьями и прокачивай словарный запас. Минималистичный дизайн, синхронизация между устройствами и полное отсутствие рекламы.

---

## 🎯 Режимы игры

| Режим | Описание |
|-------|----------|
| 🔵 **Классика** | Угадай слово за 6 попыток — идеально для ежедневной разминки |
| 🟠 **Хардкор** | Используй ВСЕ подсказки с прошлых ходов. Никаких поблажек |
| 🟢 **Рандом** | Слово случайной длины в каждой игре — разбуди в себе универсала |
| 🟣 **Дружеский вызов** | Загадай слово и брось вызов другу. Кто окажется хитрее? |

---

## ✨ Ключевые фичи

- 📖 **Встроенный словарь** — узнавай значение каждого слова прямо после игры
- 👤 **Профиль** — аккаунт, аватар, никнейм и синхронизация прогресса между устройствами
- 🏆 **Достижения** — собирай награды и открывай новые уровни сложности
- 🎚 **Гибкие настройки** — тема, клавиатура, язык интерфейса
- 🚫 **Без рекламы** — полностью, навсегда

## 🛠 Стек технологий

- **Kotlin** + **Jetpack Compose** — UI и бизнес-логика
- **MVVM** + **Hilt** — архитектура и DI
- **Room** — локальная база данных
- **Supabase** — бэкенд, авторизация, синхронизация

## 🚀 Запуск проекта

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/your-username/wordy-fresh.git
   ```
2. Открой в **Android Studio**
3. Создай файл `local.properties` и добавь ключи Supabase:
   ```
   SUPABASE_URL=your_url
   SUPABASE_KEY=your_key
   ```
4. Запусти на эмуляторе или устройстве (**minSdk 26**)
